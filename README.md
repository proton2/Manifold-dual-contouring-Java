# Manifold dual contouring Java LWJGL

Java version of Manifold Dual contouring author John Lin20 https://github.com/Lin20/isosurface

## Build Instructions
The dependencies are:
  * Maven
  * Java 11
  * I used IntelliJ IDEA Community edition
